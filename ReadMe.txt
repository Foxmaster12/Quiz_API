Данная инструкция по эксплуатации прилагается к проекту Test_task_API

После клонирования репозитория с GitHub откройте проект в подходящей среде разработки(например PyCharm), либо откройте консоль и перейдите в папку проекта(Test_task_API). Из этой папки перейдите в папку quiz. Далее либо в терминале PyCharm, либо в консоли пропишите следующие команды:
1. python install -r requirements.txt
2. python manage.py makemigrations
3. python manage.py migrate
4. python manage.py runserver
5. python manage.py createsuperuser, затем введите login и пароль для администратора
Затем перейдите по указанному URL адресу(по умолчанию: http://127.0.0.1:8000).

РАБОТА СО СТОРОНЫ АДМИНИСТРАТОРА:

1. Для перехода в панель администратора используйте URL: http://127.0.0.1:8000/admin/
2. Для аутентификации в системе в качестве администратора используйте логин и пароль созданного вами администратора.
3. Для создания дополнительного пользователя с правами администратора используйте команду(в терминале PyCharm или в консоли): python manage.py createsuperuser
3.1 Также пользователя можно создать и в панели администратора, для этого в разделе 'AUTHENTIFICATION AND AUTHORIZATION' перейдите в поле 'Users'/'add User', либо в основном разделе в поле 'Users', кнопка 'add', для изменения прав пользователя Change - выбрать нужного пользователя - изменить его настройки
4. Для создания опросов перейдите в раздел MAIN ELEMENTS и выберите поле 'Quizes'. Для создания опроса выберите опцию 'ADD QUIZ' в правом верхнем углу. Для добавления вопроса в опрос в разделе 'QUIZ-QUESTION RELATIONSHIPS' выберите вопрос из созданных ранее или создайте новый, нажав на знак плюса, который находится правее окна для вопроса. Для создания дополнительного окна для вопроса нажмите на 'Add another Quiz-question relationship'. Для сохранения готового опроса нажмите save.
5. Для изменения опросов выберите поле 'Quizes'. Выберите необходимый опрос, нажав на один из них в поле 'NAME'. После изменения параметров вопроса нажмите на поле 'Save'.
6. Для удаления опроса нажмите на поле, которое находится левее имени опроса, после появления в этом поле галочки в поле 'Action' выберите 'Delete selected Quizes' и нажмите 'Go', либо нажмите на имя опроса, а затем нажмите на кнопку 'Delete'.
7. Для работы с вопросами перейдите в раздел 'Questions'. Добавление, изменение и удаление вопросов происходит аналогичным образом.
8. Для работы с ответами перейдите в раздел 'Answers'. Добавление, изменение и удаление вопросов происходит аналогичным образом.

РАБОТА СО СТОРОНЫ ПОЛЬЗОВАТЕЛЯ:

1. Основной раздел для работы с API располагается по URL: http://127.0.0.1:8000/main/
2. Для получения списка доступных опросов перейдите по ссылке Get available quiz list, либо по URL: http://127.0.0.1:8000/main/get_quiz/
3. Для анонимного прохождения опроса нажмите на ссылку 'Pass anonymously' или по URL:http://127.0.0.1:8000/main/pass_quiz/name/anon/, где name - имя опроса. Чтобы завершить опрос нажмите на поле 'Отправить'
4. Для прохождения опроса от имени пользователя, необходима регистрация. Для этого от имени администратора создайте пользователя, затем перейдите по ссылке 'Pass quiz' или по URL:http://127.0.0.1:8000/main/pass_quiz/name/, где name - имя опроса. После этого произойдет переадресация на страницу регистрации, введите логин и пароль пользователя, затем проходите опрос.
5. Для получения пройденных пользователем опросов перейдите по ссылке 'Get passed quiz list', либо по URL: http://127.0.0.1:8000/main/get_complited_quiz/
6. Для возвращения в главное меню перейдите по ссылке 'Return to the main page' или по URL: http://127.0.0.1:8000/main/
